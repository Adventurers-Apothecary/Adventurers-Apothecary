# Adventurer's Apothecary
We co-created an e-commerce shop for selling herb seeds, planting kits, and books in a week. The project features route protection, gatekeeping middleware, and an administrative dashboard.

## Technologies used:

- [PostgreSQL](https://www.postgresql.org)
- [Sequelize](https://sequelize.org)
- [Node.js](https://nodejs.org/en/)
- [Express](https://expressjs.com)
- [JSON Web Token](https://jwt.io)
- BCrypt
- [React](https://reactjs.org)
- [Redux](https://redux.js.org)
- [Render](https://render.com/)
- [Bit.io](https://bit.io/)


## Installation
1. Install NPM packages `npm install`
2. Create database `createdb adventurers-apothecary` and seed file `npm run seed`
3. Start server `npm start` and open [http://localhost:8080](http://localhost:8080) to view it in your browser
